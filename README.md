THE DATA USED FOR THIS PROJECT WAS SORCED FROM https://www.kaggle.com/datasets/ninzaami/loan-predication
The data includes around 615 records of loan requests and status. This includes features such as their loan_id, income, gender, martial status and so on
The data includes a target variable of loan status

The objective of this project is to be able to make highly accurate predictions of the loan status on new data given a set of features
